
# Simple ToDo



## Run Locally

Clone the project

```bash
  git clone https://github.com/GhAbd/to-do.git
```

Go to the project directory

```bash
  cd to-do
```

Install dependencies

```bash
  yarn install
```

Install pods (ios)

```bash
  npx pod-install
```

Start the server

```bash
  npx react-native run-ios
```
